## 🚀 How to Run

```bash
# Clone the repo
git clone https://github.com/your-username/daily-python-programs.git

# Navigate to any day
cd daily-python-programs/Day01

# Run any program
python even_odd.py
```

> Make sure Python 3.x is installed on your system.
